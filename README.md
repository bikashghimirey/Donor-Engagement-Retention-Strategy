

# Donor Engagement & Retention Strategy Optimization

##  Project Overview
This project analyzes donor behavior for a nonprofit organization (Reaching for Rainbows) to identify strategies for improving donor retention, engagement, and long-term funding sustainability.

##  Problem Statement
- High reliance on one-time donors (~78%)
- Low donor retention (~22%)
- Strong seasonal dependency (Oct–Dec)
- Limited structured donor communication

##  Key Contributions
- Performed end-to-end data analysis on donor behavior
- Identified key patterns: retention gap, seasonality, revenue concentration
- Developed data-driven insights and visual storytelling
- Designed strategic, consulting-style recommendations
- Integrated survey insights, data, and research (triangulation)

##  Key Insights
- Donor retention is the biggest challenge
- Revenue depends on a small group of donors
- Donations are highly seasonal
- Engagement drives higher donations

##  Recommendations
- Structured donor engagement journey
- Impact-driven communication strategy
- Year-round engagement beyond campaigns
- High-value donor targeting

##  Tools Used
- Python / Jupyter Notebook
- Excel
- Data Visualization
- PowerPoint (Consulting-style presentation)

##  My Role
Led data analysis, insight generation, and strategic recommendations development.

##  Impact
This project demonstrates how data can be used to improve donor retention and create sustainable fundraising strategies.
